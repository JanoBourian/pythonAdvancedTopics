## Basic types

First of all is important remember all built-in types by default:

* Text: str
* Numeric: int, float, complex
* Sequence: list, tuple, range
* Mapping: dict
* Set: set, frozenset
* Boolean: bool
* Binary: bytes, bytearrays, memoryview
* None Type: None
